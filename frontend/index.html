Server Monitoring System

Docker Monitoring Stack

Prometheus + Node Exporter + Grafana + Nginx

Prometheus

Збирає метрики з Node Exporter і надає інтерфейс для запитів.

http://localhost:9090

Node Exporter

Метрики CPU, пам’яті, диска та мережі.

http://localhost:9100/metrics

Grafana

Готові дашборди для візуалізації метрик з Prometheus.

http://localhost:3000

Static frontend

Ця сторінка працює через Nginx.

http://localhost
Швидкий старт:
1) ./prepare-app.sh 2) ./start-app.sh 3) ./stop-app.sh 4) ./remove-app.sh
Якщо використовуєте `env_file`, додайте `.env` до .gitignore.
